# Skill: Status Report

Use this format whenever reporting milestone updates or sprint completions in Slack channels or files.

## Template Format

```markdown
### 📊 Status Report

#### ✅ What I Did
- [List items completed during the session/sprint]

#### 🕒 What's Left
- [List pending sprint backlog items]

#### ⚠️ What Needs Your Call
- [List key architectural blocks, design questions, or approvals required]
```
